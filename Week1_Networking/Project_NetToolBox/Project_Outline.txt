## ✅ Final Toolkit Plan: *"NetToolbox: DevOps Network Troubleshooting CLI"*

### 🎯 Project Goal:

A single Python CLI application that:

* Groups and logically categorizes overlapping tools
* Explains each command when run
* Interactively takes user input where needed (e.g., domain/IP)
* Shows formatted output + saves to logs
* Helps users **learn** while they **troubleshoot**

---

## 🧠 Updated Command List (Grouped Logically)

| Category                | Commands Included                                                       |
| ----------------------- | ----------------------------------------------------------------------- |
| **Basic Connectivity**  | `ping`, `traceroute/tracepath`, `mtr`, `telnet`, `nc`                   |
| **DNS & IP Tools**      | `nslookup`, `dig`, `whois`, `hostname`, `host` (auto include if needed) |
| **Interface & IP Info** | `ifconfig`, `ip`, `iwconfig`, `ifplugstatus`, `route`                   |
| **Firewall & Ports**    | `iptables`, `nmap`, `netstat`, `ss`                                     |
| **File/Content Tools**  | `wget`, `curl`, `jq`                                                    |
| **Utility/Monitoring**  | `watch`, `arp`                                                          |

✅ **Dropped duplicates**:

* `traceroute` vs `tracepath` → we'll use just one based on OS support
* `nc` covers basic `telnet` functions
